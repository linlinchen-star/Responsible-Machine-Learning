Individual Homework 4 - Robustness, Generalization, and Dataset Drift Audit

## The purpose of the analysis
Evaluate whether the COMPAS recidivism prediction models remain reliable beyond the training setting by auditing distribution drift, generalization, spurious correlations, robustness under stress, and slice-based performance disparities.
## The Python libraries used
`pandas` | `numpy` | `scipy` | `scikit-learn` | `matplotlib`
## Instructions for reproducing the results
Download and open `ASSN4_rML.ipynb` in Jupyter or Google Colab and run all cells in order. The notebook is self-contained: it downloads the COMPAS data directly from the ProPublica GitHub repository, so no local data file is needed.
