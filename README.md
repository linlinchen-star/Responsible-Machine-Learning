Individual Homework 1 - Translating an R Machine Learning Workflow into Python

## The purpose of the analysis
The purpose of this analysis is to reproduce the analytical workflow from Lecture 01 using Python. This includes performing exploratory data analysis (EDA), building a machine learning model using logistic regression, and conducting model diagnostics. The goal is to ensure that every substantive step from the original R script is implemented in an equivalent Python form.
## The Python libraries used
`pandas` |  `numpy` | `scipy` | `statsmodels` |
## Instructions for reproducing the results
Download and open `ASSN1_rML.ipynb` in Jupyter or Google Colab and run all cells in order. The notebook is self-contained: it downloads the COMPAS data directly from the ProPublica GitHub repository, so no local data file is needed.
