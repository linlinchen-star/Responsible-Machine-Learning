Individual Homework 5 - ML Security Audit

## The purpose of the analysis
Audit the fairness and security robustness of ML models trained on the ProPublica COMPAS dataset by simulating three classes of adversarial attacks: PGD evasion, label-flip poisoning, and membership inference. Evaluate their impact on model accuracy and racial fairness metrics.
## The Python libraries used
`pandas` | `numpy`| `scikit-learn` | `matplotlib`
## Instructions for reproducing the results
Download and open `ASSN5_rML.ipynb` in Jupyter or Google Colab and run all cells in order. The notebook is self-contained: it downloads the COMPAS data directly from the ProPublica GitHub repository, so no local data file is needed.
